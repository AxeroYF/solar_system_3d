# 3D 太阳系模拟

基于 Three.js 的交互式 3D 太阳系可视化项目。支持行星公转/自转模拟、真实比例切换、时间旅行等功能。

## 功能特性

- **完整的太阳系天体**：太阳、八大行星、矮行星（谷神星、冥王星）、哈雷彗星、国际空间站及多颗卫星
- **真实轨道力学**：基于 J2000 历元的开普勒椭圆轨道，牛顿迭代求解偏近点角
- **时间控制**：0.1x ~ 365x 变速、暂停/播放、日期跳转（手动输入年月日）
- **真实比例模式**：一键切换天体大小和轨道距离为真实比例
- **交互操作**：鼠标/触屏旋转缩放平移，点击天体查看详细信息和 3D 预览
- **视觉效果**：大气光晕着色器（地球/金星/火星）、土星环纹理、地球云层、哈雷彗尾
- **小行星带 & 柯伊伯带**：粒子系统模拟

## 快速开始

本项目使用 ES Modules，需要通过 HTTP 服务器访问（不支持直接双击 HTML 打开）。

**推荐方式：双击启动脚本**

```
双击 start.bat
```

脚本会自动启动本地 Python 服务器，并在浏览器打开项目页面：

```
http://localhost:8000/solar_system.html
```

**手动方式：启动本地服务器**

```bash
# Python
python -m http.server 8000

# 或 Node.js
npx serve
```

然后在浏览器访问：

```
http://localhost:8000/solar_system.html
```

注意：不要直接双击 `solar_system.html`，否则浏览器可能会因为模块加载限制导致页面无法正常运行。

## 项目结构

```
solar_system_3d/
├── solar_system.html  # HTML 页面结构 + import map
├── style.css          # 全部 CSS 样式
├── start.bat          # 一键启动脚本（Windows）
├── js/
│   ├── main.js        # 入口文件，初始化上下文并组装各模块
│   ├── data.js        # 行星数据、纹理 URL、着色器、常量
│   ├── scene.js       # 场景/相机/渲染器/控制器/灯光/太阳/星带
│   ├── planets.js     # 行星创建函数及轨道力学计算
│   ├── ui.js          # UI 面板交互、聚焦、预览、时间/比例控制
│   └── animation.js   # 主动画循环
└── textures/          # 本地纹理资源（备用）
```

## 技术栈

- [Three.js](https://threejs.org/) v0.160.0 — 3D 渲染
- [Tween.js](https://github.com/tweenjs/tween.js) v23.1.1 — 相机过渡动画
- 自定义 GLSL 着色器 — 大气层光晕效果
- ES Modules + Import Maps — 零构建工具的模块化架构
