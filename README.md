主要是做了一些星球轨迹的修正：
*   现在星球的轨迹被正确显示为椭圆形了。
*   卫星的轨迹也不完全在一个平面上了。
*   一些触屏的优化、木星和土星的卫星的干涉问题。
*   小行星带和遥远的冥王星也加入系统了。

*   添加国际空间站、谷神星、柯依柏带。
